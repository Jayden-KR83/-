# Scoring Criteria Skill

## 역할
CDP 제출 답변을 채점 기준에 따라 자동 평가하고 감점 원인과 개선 방향을 제시한다.
Claude API를 통해 맥락 이해 기반의 정성적 채점을 수행한다.

## 채점 원칙
1. 각 문항의 max_points 기준으로 0~max_points 점수 부여
2. 감점 사유는 구체적으로 명시 (단순 "부족" 금지)
3. 개선 제안은 실행 가능한 수준으로 작성
4. confidence: 낮음/보통/높음 표시

## 채점 카테고리
- 완전성(Completeness): 요구 정보가 모두 포함되었는가
- 정확성(Accuracy): 수치, 단위, 기준연도가 정확한가
- 일관성(Consistency): 다른 문항과 데이터가 일치하는가
- 구체성(Specificity): 일반론이 아닌 회사 특정 내용인가

## 감점 기준
| 유형 | 예시 | 감점 범위 |
|------|------|-----------|
| 필수 항목 누락 | Scope1 수치 없음 | -30~50% |
| 단위 오류 | MtCO2e vs tCO2e 혼용 | -10~20% |
| 기준연도 불일치 | 전사 기준연도와 다름 | -15~25% |
| 근거 자료 미제시 | 감축 목표 수치 출처 없음 | -10~20% |

## 주의사항
- 채점 기준 변경 시 이 파일만 업데이트하면 Agent 즉시 반영
- Claude API 호출 시 답변 원문 전체 전송 전 민감정보 포함 여부 확인
