# Draft Writing Skill

## 목적
CDP 문항별 답변 초안을 자동 생성한다.
크롤링된 ESG 외부 데이터 + 회사 Reference 데이터를 종합해 고득점 답변 구조로 작성한다.

## 답변 작성 원칙
1. 문항 ID와 배점을 반드시 확인하고 배점 비중에 맞게 상세도 조절
2. 회사 특정 데이터(수치, 목표, 연도)를 반드시 포함 — 일반론 금지
3. CDP 공식 채점 4개 카테고리 모두 커버
   - 완전성: 필수 항목 누락 없이
   - 정확성: 수치·단위·기준연도 명시
   - 일관성: 타 문항과 데이터 충돌 없이
   - 구체성: 회사 이름, 사업장, 연도 특정
4. 출처 인용: 크롤링 데이터 활용 시 출처 URL 또는 문서명 명시
5. 한국어 기본, CDP 공식 용어(영문) 병기

## 답변 구조 템플릿
```
[개요]
(1-2문장: 핵심 답변 요약)

[상세 내용]
(문항 요구사항별 항목화)
- 항목 1: 수치/내용
- 항목 2: 수치/내용

[근거 및 출처]
(데이터 출처, 보고서명, 측정 방법론)

[향후 계획]
(목표, 일정, 담당 조직)
```

## 데이터 활용 우선순위
1. 회사 내부 데이터 (vector_store에 저장된 reference 문서)
2. 크롤링된 CDP 공식 가이드라인
3. 업계 벤치마크 (SBTi, GHG Protocol 기준)

## 품질 기준
- 최소 200자 이상 답변
- 수치 데이터 최소 1개 이상 포함
- 근거 출처 최소 1개 이상 명시
- confidence 판정: 높음(내부데이터 있음) / 보통(외부데이터만) / 낮음(데이터 없음)

## 주의사항
- 확인되지 않은 수치 추정 금지 — 데이터 없으면 "[데이터 필요]" 표기
- 경쟁사 비교 내용 포함 금지
- 규제 위반 가능성 있는 내용 포함 금지
