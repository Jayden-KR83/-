# Validator Skill

## 역할
Agent 출력의 품질을 검증하고, 기준 미달 시 재시도 또는 사람 검토를 요청한다.

## 검증 단계
### 1단계: 구조 검증
- 필수 필드 존재 여부, 데이터 타입, 범위 초과 값

### 2단계: 품질 검증
- 빈 응답, "모름"/"해당없음" 탐지
- 언어 일관성 검사

### 3단계: 재시도 판단
| confidence_score | 조치 |
|-----------------|------|
| >= 0.85 | 정상 반환 |
| 0.65 ~ 0.84 | 경고 포함 반환 (사람 검토 권장) |
| < 0.65 | 자동 재시도 (최대 2회) |
| 재시도 후 미달 | needs_review 반환 |

## confidence_score 계산
- 필수 필드 완성도: 40%
- 텍스트 길이 적절성: 20%
- 수치 데이터 존재: 20%
- 언어 일관성: 20%
