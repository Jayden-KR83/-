워크벤치란?

CDP(Carbon Disclosure Project) 설문 응답을 최대한 자동화하는 도구입니다.
담당자가 매년 수백 개의 문항을 수작업으로 작성하는 대신, AI가 초안을 생성하고 담당자는 검수·수정만 하도록 설계됐습니다.

📋 입력

cdp_master.xlsx — CDP 공식 Comm. Tool SKEP 시트.
문항 정의, 채점 기준, 전년도 답변이 모두 포함된 마스터 파일.

📤 출력

BX열(2025 CDP 1st_Ans.) 자동 채우기.
최종 확정 답변은 CB열(Final_Ans.)에 저장.
처리 결과 요약 통계 + 검수 목록 제공.

목표 자동화율

Phase 1 — 규칙 기반 Close 스킵 + 전년도 복사 → 약 40~50%
Phase 2 — AI 초안 Claude API 생성 포함 → 약 70~80%

단계담당설명
1. Excel 업로드 담당자 data/reference/ 폴더에 저장. 최초 1회 또는 파일 교체 시
2. 문항 로드 ExcelDataLoader Comm. Tool_SKEP 시트 4행부터 파싱 → CDPQuestion 객체 리스트 생성
3. 유형 분류 AnswerTypeClassifier BV열(Sub_Q_Op) 패턴 분석 → 9가지 답변 유형 + 3단계 자동화 레벨 결정
4. Phase 1 RuleBasedEngine Close(X) 스킵, 전년도 복사, 고정값 적용. 비용 0원, 즉시 처리
5. Phase 2 AIAnswerGenerator Phase 1에서 처리 못한 문항을 Claude API로 초안 생성 (enable_ai=True 시)
6. 검수 담당자 워크벤치 UI에서 문항별로 AI 초안 확인·수정
7. BX/CB 저장 ExcelSaver 확정 답변을 원본 Excel BX열 또는 CB열에 직접 기록

핵심 컴포넌트

📦 CDPQuestion

문항 1행 = 1 객체.
질문 텍스트, 선택지, 채점 기준, 전년도 답변, 현재 답변 등 모든 필드 포함.

backend/agents/cdp_auto_answer_module.py

🔍 AnswerTypeClassifier

BV열의 텍스트 패턴으로 답변 유형 분류.
e.g. "Select all that apply" → MULTI_SELECT

classify() + determine_auto_level()

⚙️ RuleBasedEngine

Phase 1 규칙 엔진.
Close 스킵, 전년도 복사, 회사 고정값(LEI 등).
신뢰도 1.0, 비용 0원.

🤖 AIAnswerGenerator

Phase 2 AI 엔진.
Claude에게 질문+채점 기준+전년도 답변을 넘겨 초안 생성.

Excel 파싱 전담 클래스.
COL_MAP 기반으로 열 인덱스를 필드에 매핑.
XLS/XLSX 자동 감지 변환 포함.

🎛️ CDPAutoAnswerOrchestrator

Phase 1→2 순서 제어.
문항별로 Phase 1 시도 → 실패 시 Phase 2 → 결과 집계.

답변 유형 분류 (AnswerType)

BV열(영문 선택지) 텍스트 패턴을 정규식으로 분석합니다.

유형패턴 예시처리 특성
SINGLE_SELECT "Select from: ..." 한 개 선택. 전년도 복사 가능
MULTI_SELECT "Select all that apply" 여러 개 선택. 전년도 복사 가능
TEXT_SHORT "Text field [max 1500 chars]" ≤1500자 텍스트. AI 초안 대상
TEXT_LONG "Text field [max 5000 chars]" >1500자. AI 초안 + 상세 검수 필요
NUMERICAL "Numerical field", "enter a number" 숫자 입력. 전년도 복사 가능
PERCENTAGE "Percentage field" % 입력. 전년도 복사 가능
DATE "[DD/MM/YYYY]" 날짜 입력. 전년도 복사 가능
ATTACHMENT "[Attachment ..." 파일 첨부. 수작업 필수
SKIP BQ열 = "X" 또는 옵션 없음 제출 불필요 문항. 자동 스킵

자동화 레벨 (AutoLevel)

문항마다 처리 방식과 검수 필요 여부가 결정됩니다.

🟢

FULL_AUTO

신뢰도 ≥ 0.95.
담당자 검수 없이 자동 저장 가능.

대상: Close 문항, 변경 없는 선택/수치형

🟡

AI_DRAFT

AI가 초안 생성.
담당자 검수 후 저장 권장.

대상: 텍스트형, 전년도 있는 장문

🔴

MANUAL_REQUIRED

자동화 불가.
담당자가 직접 입력해야 함.

대상: 첨부파일, 전년도 없는 신규 장문

워크벤치 UI에서의 표시

레벨UI 표시BX 저장 조건
FULL_AUTO 초록색, "자동" 배지 신뢰도 ≥ 0.9 → 일괄 저장 가능
AI_DRAFT 노란색 ⚡, "검수 필요" 배지 담당자 승인 후 개별 저장
MANUAL 빨간색, "수작업" 배지 UI에서 직접 입력 후 저장

COL_MAP — 열 인덱스 매핑

Excel 열은 0-based 인덱스로 매핑됩니다. Excel 파일 구조가 바뀌면 이 매핑만 수정하면 됩니다.

COL_MAP = {
    "order":            0,   # A  순번
    "q_no":             1,   # B  Q_No.
    "level":            2,   # C  Lv.
    "q_type":           17,  # R  질문 유형
    "main_question_kr": 19,  # T  K_Q_Des. (한국어)
    "main_question":    20,  # U  Q_Des.   (영어)
    "scoring_dc":       22,  # W  DC
    "scoring_ac":       24,  # Y  AC
    "scoring_mc":       26,  # AA MC
    "scoring_lc":       28,  # AC LC
    "d_den_cdp":        31,  # AF
    ...
    "open_close":       68,  # BQ  ← X이면 스킵
    "sub_q_options":    73,  # BV  ← 유형 분류 기준
    "current_answer":   75,  # BX  ← 자동화 저장 대상
    "prev_answer":      77,  # BZ  ← 전년도 답변
    "final_answer":     79,  # CB  ← 최종 저장
}
⚠️ 열 매핑 오류 시 CDP가 Comm. Tool을 업데이트하면 열 번호가 바뀔 수 있습니다.
업로드 후 반드시 POST /cdp-answer/verify-columns API로 매핑을 검증하세요.

워크벤치 사용 순서

1

Excel 파일 업로드

CDP 워크벤치 팝업 → Excel 파일 선택 → 문항 로드 버튼.
또는 드롭다운에서 기업 업로드된 파일 선택.

2

문항 목록 확인

로드 완료 후 Q_No 검색, 유형 필터, 상태 필터로 원하는 문항 탐색.
통계 표시: 전체 / Open / 답변완료 / 미완료 / 채점항목 수.

3

(선택) Phase 1만 먼저 실행

enable_ai=False로 규칙 기반만 실행.
API 비용 없이 Close 스킵 + 전년도 복사 결과 미리 확인.

4

개별 문항 AI 초안 생성

문항 카드의 ⚡ AI 초안 생성 버튼 클릭.
Claude가 채점 기준 + 전년도 답변 + 회사 컨텍스트를 바탕으로 초안 작성.

5

초안 검수 및 수정

생성된 초안 텍스트를 편집.
채점 기준(D/A/M/L) 패널 참고하며 내용 보완.

6

BX열 저장

BX 저장 버튼 → POST /cdp-answer/save-answer API 호출.
원본 Excel BX열(2025 CDP 1st_Ans.)에 답변 직접 기록.

파일 오류 해결

❌ 로드 실패: File is not a zip file 이 오류는 업로드한 xlsx 파일이 실제로는 구형 XLS(OLE2) 포맷이거나,
Microsoft Information Protection(MIP) 감도 레이블 암호화가 적용된 파일일 때 발생합니다.

원인 1: 구형 .xls 포맷 (확장자만 .xlsx)

Excel 97-2003 형식으로 저장된 파일. 코드에서 자동 변환을 시도하지만, 일부 암호화 파일은 불가합니다.

원인 2: MIP 감도 레이블 암호화 (현재 해당)

파일 내부가 EncryptedPackage 스트림으로 래핑되어 Python이 읽을 수 없습니다.
Excel을 통해서만 복호화 가능 (Office 365 계정 인증 필요).

✅ 해결 방법

  1. Excel에서 cdp_master.xlsx 열기
  2. 파일 → 다른 이름으로 저장 클릭
  3. 파일명: cdp_master_plain.xlsx
  4. 저장 시 민감도 레이블을 "일반(General)"으로 변경 (또는 레이블 없이 저장)
  5. 저장된 파일을 워크벤치에 업로드

레이블 변경 권한이 없으면: 파일 → 내보내기 → 통합 문서 사본 만들기 로 시도

API 엔드포인트

메서드경로설명
GET /cdp-answer/excel-list data/reference/ 폴더의 Excel 파일 목록 반환
POST /cdp-answer/upload-excel Excel 파일 업로드 (multipart/form-data)
POST /cdp-answer/verify-columns COL_MAP ↔ 실제 열 헤더 정합성 검증
POST /cdp-answer/questions 전체 문항 목록 + 통계 반환
POST /cdp-answer/generate-single 개별 문항 AI 초안 생성 (row_idx 지정)
POST /cdp-answer/save-answer 답변을 Excel BX 또는 CB열에 저장
POST /cdp-answer/run 전체 문항 일괄 자동 생성 (Phase 1+2)
